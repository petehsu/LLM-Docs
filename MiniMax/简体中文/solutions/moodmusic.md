# 心情音乐生成APP

> 基于music模型，支持 Web 网页版和 Android 移动端。用户只需描述当前心情，AI 即可自动生成包含歌词和音频的原创音乐作品。通过 LLM 深度理解情绪、结构化歌词创作、可编辑工作流等创新设计，适用于情绪记录、社交分享、创意灵感等场景。

### 智能心情理解与音乐风格生成

利用 LLM分析用户的心情描述文本，自动提取情绪关键词并转换为专业的音乐制作指令。AI会智能生成包含曲风（爵士/流行/摇滚等）、人声类型（男声/女声）、乐器配置（钢琴/吉他等）、情绪氛围等多维度参数的音乐风格描述，将"我今天很开心"这样的简单表达转化为"流行,女声,吉他伴奏,温柔,午后阳光"的专业提示词，让每个人都能轻松驾驭音乐创作。

### 结构化歌词智能创作

基于 LLM 的歌词生成，自动创作符合专业音乐制作规范的结构化歌词。AI 严格按照 \[Intro]、\[Verse]、\[Pre-Chorus]、\[Chorus]、\[Outro] 等标准段落标记组织内容，节奏紧凑流畅。

### 未来更多玩法

* 输入图片生成歌曲
* 输入文章或网页链接生成歌曲
* 相同歌词不同曲风的同时抽卡
* 专业模式（当前项目做了简化）
