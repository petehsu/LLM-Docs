# 快速接入

> 参考文档，了解关于 Coding Plan 的订阅及快速接入方式

## 快速接入

仅需要3步 快速获得Coding Plan 服务能力

<Steps>
  <Step title="订阅编程套餐">
    访问 [Coding Plan](https://platform.minimaxi.com/subscribe/coding-plan) 选择最适合您的编程套餐方案，Starter **首月9.9元** 限时活动进行中
  </Step>

  <Step title="获取 Coding Plan API Key">
    访问 [账户管理/Coding Plan](https://platform.minimaxi.com/user-center/payment/coding-plan) 获取 Coding Plan 的 **API Key**
  </Step>

  <Step title="接入编程工具">
    选择您常用的 AI 编程工具，体验最新 **MiniMax M2** 能力
  </Step>
</Steps>

您也可参考以下详细步骤，快速激活您的 Coding Plan 服务

### 步骤1：订阅 Coding Plan 方案

访问 [Coding Plan](https://platform.minimaxi.com/subscribe/coding-plan) 订阅页面，根据您的需求选择合适的订阅方案（**Starter, Plus, Max**）并完成订阅。

![Coding Plan](https://filecdn.minimax.chat/public/5ca037c6-2acb-4ad5-b56d-c5eb2ca9652a.png)

### 步骤2：获取您的专属 API Key

订阅成功后，请前往 [账户管理/Conding Plan](https://platform.minimaxi.com/user-center/payment/coding-plan)  页面，查看您订阅的 Coding Plan 套餐，并获得 Coding Plan 的API Key，在编程工具中使用。

![apikey](https://filecdn.minimax.chat/public/2e4dbe4b-4782-4a42-98c5-d242617c0ab4.png)

<Callout icon="album" color="#A8A8A8" iconType="regular">
  重要提示：

  * 此 API Key 是 **Coding Plan** 专属的，它和按量计费的文本模型的 API Key 并不互通。
  * 此 API Key 仅在您订阅 **Coding Plan** 有效期内有效。
  * 请妥善保存您的 API Key 避免您的资源损失。
</Callout>
