## 二、 受邀用户福利

1. **立减优惠**：

受邀用户通过邀请链接或邀请码完成注册，并首次成功付费订阅且**仅限于订阅【GLM Coding】服务**时，可享受该订单金额10%的立减优惠（立减金额将直接从订单实付总额中扣除）。

2. **适用对象**：

* 仅限**平台新注册用户**，或**已注册但从未有过任何付费订阅记录的历史用户**。
* 每个用户（以手机号、邮箱为准）仅可以**受邀者身份享受一次**首单立减优惠。

3. **除外情形**：

* 该优惠仅适用于**首次订阅**的当前订单。
* **后续的续订、续费、升级、降级或其他任何形式的后续订阅订单，均不再享受此立减优惠**。
* 该优惠不可与平台其他同类首单优惠活动叠加享受。

## 三、 邀请者福利

邀请用户（以下简称"邀请人"）成功邀请好友（即"受邀用户"）完成注册并完成首单有效支付后，可获得以下赠金奖励：

1. **有效邀请的时效性**：

好友通过推广链接进入并在活动页面完成登录可建立绑定关系，**72小时内**完成首次订阅支付邀请人可获得相应赠金奖励。若好友同时点击多个邀请链接，产生的奖励归属于有效期内最后一次触达的邀请人。

2. **基础奖励**：

* 每成功邀请1名有效好友，邀请人可获得该好友**首单实际支付金额**（定义见下文）的10%作为赠金。
* 邀请人数无上限，奖励可持续累积。

3. **阶梯额外奖励**：

* 邀请人**每累计成功邀请满30名有效好友**，即可额外获得这30名好友**首单实际支付总额**的10%作为一次性额外赠金。
* 额外奖励以"**轮**"为单位计算，每满30人即触发一轮，可循环累积，上不封顶。

4. **"实际支付金额"定义**：

* 指受邀用户首单订阅时，**最终实际支付的现金金额（最低0.1元），包括账户现金余额、第三方支付的金额**。
* **不包括**因使用活动优惠、赠金等任何优惠减免部分。

5. **赠金性质与使用说明**：

* **性质**：赠金是平台向用户提供的**专属优惠权益**，不具有现金价值。
* **用途**：赠金仅可用于在大模型开放平台抵扣以下消费：
  * 购买或续费**GLM Coding**及其他订阅服务。
  * 购买平台内的资源包、功能扩展包等虚拟产品。
  * 支付平台API调用产生的费用。
* **限制**：赠金**不可提现、不可转让、不可赠送**，也**不可用于要求订阅服务的退款**。
* **有效期**：赠金**长期有效**，除非用户账号注销或平台因故终止运营。但平台保留在未来根据业务情况调整赠金有效期的权利，如调整将提前30天公告通知。
