# Claude Code IDE 插件

> Claude Code 结合 GLM Coding Plan 在 Vscode, Jetbrains IDE 的插件的使用方法

Claude Code 是一个智能编码工具，可以在终端中运行，也可以通过在 Vscode, Jetbrains 等 IDE 安装插件使用。

<Tip>
  搭配 [**GLM Coding Plan**](https://zhipuaishengchan.datasink.sensorsdata.cn/t/Nd)，Claude Code 的能力能进一步增强 —— 以更低价格获得 **3 倍用量**，让你在编码、调试和工作流管理中更高效、更稳定。
</Tip>

## 步骤一：安装 Claude Code & 配置 GLM Coding Plan

<Tip>
  Claude Code IDE 插件依赖 Claude Code CLI 工具，请先参考 [**Claude Code 使用 GLM Coding Plan 指南**](./claude)  完成 Claude Code 的安装与 GLM Coding Plan 的配置。
</Tip>

参考 [**Claude Code 使用 GLM Coding Plan 指南**](./claude)  完成 Claude Code 的安装与 GLM Coding Plan 的配置

## 步骤二：安装使用 IDE 插件

<Tabs>
  <Tab title="Claude Code for VS Code 插件">
    <Info>
      Claude Code Vscode 插件支持在 Vscode 及基于 Vscode 的 IDE（如 Cursor、Trae 等）中使用。
    </Info>

    打开 Vscode，进入扩展市场，搜索并安装「Claude Code」插件进行安装。

    ![Description](https://cdn.bigmodel.cn/markdown/1763349262442image.png?attname=image.png)

    安装完成后，点击如下图中的右上角 Claude Code 图标，即进入 Claude Code 页面，在登录页面等待几秒待其初始化完成后即可使用。

    ![Description](https://cdn.bigmodel.cn/markdown/1763349672819image.png?attname=image.png)

    成功进入 Claude Code 对话框后，在对话框输入 `/config` 进入设置，勾选 `Disable Login Prompt` 配置来关闭登录页面。

    ![Description](https://cdn.bigmodel.cn/markdown/1765784577084image.png?attname=image.png)
  </Tab>

  <Tab title="Claude Code for Jetbrains 插件">
    <Info>
      Claude Code Jetbrains 插件支持 Jetbrains 的系列 IDE 如 IntelliJ IDEA、PyCharm、WebStorm 等。
    </Info>

    打开 Jetbrains IDE（如 IntelliJ IDEA、PyCharm 等），进入插件市场，搜索「Claude Code」插件并进行安装。

    ![Description](https://cdn.bigmodel.cn/markdown/1763349847970image.png?attname=image.png)

    安装完成后，重启 IDE 即可使用 Claude Code 功能。

    ![Description](https://cdn.bigmodel.cn/markdown/1763350722159image.png?attname=image.png)
  </Tab>
</Tabs>

<Tip>
  完毕！现在就可以在 IDE 中正常使用 Claude Code 进行开发了。
</Tip>
