## 2. 知识库存储计费规则

| 类别     | 子项      | 详细内容                                                                                                             |
| ------ | ------- | ---------------------------------------------------------------------------------------------------------------- |
| 计费规则详情 | 免费额度    | <ul><li>我们为所有用户提供永久免费的存储额度</li><li>存储容量：1 GB</li><li>只要您的知识库总存储量在 1 GB 以内，即可永久免费使用，无时间限制</li></ul>               |
|        | 超额计费    | <ul><li>当您的知识库存储量超出 1 GB 的免费额度时，超出的部分将按以下规则计费</li><li>计费单价：0.04 元/GB/小时</li><li>计费方式：按量计费，后付费（先使用，后结算）</li></ul> |
|        | 计费与结算细则 | <ul><li>计费周期：我们以小时为单位进行计费和结算</li><li>账单生成：系统每小时自动生成账单，并从您的账户余额中扣除相应费用；您可以随时在后台查看详细的账单记录</li></ul>                |

| 类别     | 子项            | 详细内容                                                                                                                                                                                                                                                                                        |
| ------ | ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 欠费处理规则 | 第一阶段：欠费 1–7 天 | <ul><li>用户影响：知识库将被暂停服务，无法上传、调用或使用知识库内的任何内容</li><li>系统处理：您的数据会被安全保留；系统会通过短信等方式每两日提醒您账户已欠费</li><li>如何恢复：在这 7 天内结清所有欠费账单，服务将自动恢复，数据完好无损</li></ul>                                                                                                                                              |
|        | 第二阶段：欠费超过 7 天 | <ul><li>您的数据将面临被删除的风险</li><li>用户影响：知识库服务持续暂停，且存储的数据将无法恢复</li><li>系统处理：为了释放存储资源，系统会将您的知识库数据列入删除计划；系统将为您保留最近上传的 1 GB 数据，并删除超出 1 GB 的部分；在数据被删除前的 24 小时，我们会向您发送一次通知，提醒您充值以避免数据丢失</li><li>若您未在此之前完成充值，您的数据将被永久删除</li><li>如何恢复：在数据删除前结清欠款，服务可恢复；在数据被删除后，即使您补缴了欠款，账户可以恢复正常状态，但已删除的数据无法找回</li></ul> |
