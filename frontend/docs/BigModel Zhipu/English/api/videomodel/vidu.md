## Vidu

High-performance video large model with exceptional consistency and dynamic capabilities. Features precise semantic understanding and outstanding inference speed. Supports text-to-video, image-to-video, start-end frame generation, and multi-subject reference-based video generation, empowering full-spectrum video creation.

| Capability | **Description** |
| --- | --- |
| Text-to-Video | Generates videos from text input; currently supports general styles and specially optimized anime styles. |
| Image-to-Video | Generates videos using an initial frame image and corresponding text description. |
| Start-End Frames | Accepts two images: the first as the start frame and the second as the end frame. The model generates videos based on these parameters. |
| Reference-Based Video | Generates videos using reference images and text descriptions. Supports multiple subjects like characters and objects. |

* **Model Codes**: viduq1-text, viduq1-image, viduq1-start-end, vidu2-image, vidu2-start-end, vidu2-reference
* **Compare Models**: [Vidu Series Differences](https://bigmodel.cn/dev/howuse/model)
* **Pricing**: [Product Pricing](https://www.bigmodel.cn/pricing)
* **Rate Limits**: [Rate Limits](https://www.bigmodel.cn/dev/howuse/rate-limits)
* **API Key**: [Manage API Keys](https://www.bigmodel.cn/usercenter/proj-mgmt/apikeys)

## Video Generation Task

### Interface Request

| Transmission Method | https |
| --- | --- |
| Request URL | `https://open.bigmodel.cn/api/paas/v4/videos/generations` |
| Call Method | Asynchronous; results require querying. |
| Character Encoding | UTF-8 |
| Request Format | JSON |
| Response Format | JSON |
| Request Type | POST |

---

### Request Parameters

#### Text-to-Video

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `model` | String | Yes | Model code.  Options: `viduq1-text` |
| `style` | String | No | Style . Default: `general`. Options: `general`, `anime`. - `general`: Universal style, controllable via prompts. - `anime`: Anime-optimized style. |
| `prompt` | String | Yes | Text description (‚â§1500 characters). |
| `duration` | String | No | Video length. Default model-dependent:  - `viduq1-text`: 5s (only). |
| `aspect_ratio` | String | No | Aspect ratio.  Default: `16:9`. Options: `16:9`, `9:16`, `1:1`. |
| `size` | String | No | Resolution.  Default model-dependent:  - `viduq1-text` (5s): `1920x1080` (only). |
| `movement_amplitude` | String | No | Motion amplitude. Default: `auto`. Options: `auto`, `small`, `medium`, `large`. |
| `request_id` | String | No | Unique user-provided request ID. Generated by platform if omitted. |
| `user_id` | String | No | End-user unique ID (6-128 chars) for security audits. Learn more. |

---

#### Image-to-Video

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `model` | String | Yes | Model code.  Options: `viduq1-image`, `vidu2-image`. |
| `image_url` | Object | Optional | Initial frame image (URL/Base64).  Notes:  1. Supports 1 image (PNG/JPEG/JPG/WEBP).  2. Aspect ratio ‚â§1:4 or 4:1; size ‚â§50MB.  3. Base64 format: `data:image/png;base64,{base64_encode}`. |
| `prompt` | String | Optional | Text description (‚â§1500 chars). Either `image_url` or `prompt` must be provided. |
| `duration` | String | No | Video length. Default model-dependent:  - `viduq1-image`: 5s - `vidu2-image`: 4s. |
| `size` | String | No | Resolution. Default model-dependent:  - `viduq1-image` (5s): `1920x1080`  - `vidu2-image` (4s): `1280x720` `480x360` |
| `movement_amplitude` | String | No | Motion amplitude.  Default: `auto`. Options: `auto`, `small`, `medium`, `large`. |
| `with_audio` | bool | No | Add background music.  Default: `false`. Only supports 4s videos. |
| `request_id` | String | No | Unique request ID. |
| `user_id` | String | No | End-user unique ID. |

---

| `size` | String | No | Resolution. Default model-dependent:  - `viduq1-start-end` (5s): `1920x1080`  - `vidu2-start-end` (4s): `1280x720` `480x360`. |
| `movement_amplitude` | String | No | Motion amplitude. Default: `auto`. Options: `auto`, `small`, `medium`, `large`. |
| `with_audio` | bool | No | Add background music. Default: `false`. Only supports 4s videos. |
| `request_id` | String | No | Unique request ID. |

| `prompt` | String | Yes | Text description (‚â§1500 chars). Either `image_url` or `prompt` must be provided. |
| `duration` | String | No | Video length. Default: 4s. |

## Task Result Query

### Interface Request

| Transmission Method | https |
| --- | --- |
| Request URL | `https://open.bigmodel.cn/api/paas/v4/async-result/{id}` |
| Call Method | Synchronous (waits for task completion). |
| Character Encoding | UTF-8 |
| Request Format | JSON |
| Response Format | JSON |
| Request Type | GET |

### Request Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | String | Yes | Task ID. |

### Response Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `model` | String | Model name. |
| `video_result` | List | Generated video results. |
| `url` | String | Video URL. |
| `cover_image_url` | String | Video cover image URL. |
| `task_status` | String | Status: `PROCESSING`, `SUCCESS`, `FAIL`. |
| `request_id` | String | Request ID. |

```
from zhipuai import ZhipuAI
client = ZhipuAI(api_key="") # Fill in your APIKey
response = client.videos.retrieve_videos_result(
    id="8868902201637896192"
)
print(response)
```
