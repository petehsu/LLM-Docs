# Kimi K2 模型

一款具备超强代码和 Agent 能力的 MoE 架构基础模型，总参数 1T，激活参数 32B，上下文最长 256k。在通用知识推理、编程、数学、Agent 等主要类别的基准性能测试中，K2 模型的性能超过其他主流开源模型
