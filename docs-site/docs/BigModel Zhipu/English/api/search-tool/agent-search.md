Based on search intent, it intelligently decomposes multiple search queries, retrieves search results, and synthesizes all webpage content to generate a summarized model response. The answers are comprehensive and in-depth.

## API Request

| Attribute | Value |
| --- | --- |
| Request URL | https://open.bigmodel.cn/api/paas/v4/assistant |
| Calling Method | Supports only SSE calls |
| Character Encoding | UTF-8 |
| Request Format | JSON |
| Response Format | JSON or standard Stream Event |
| Request Type | POST |
| Development Language | Any language capable of HTTP requests |

### Request Parameters

| Parameter Name | Type | Required | Description |
| --- | --- | --- | --- |
| **assistant\_id** | String | Yes | AI Search Agent ID: `659e54b1b8006379b4b2abd6` |
| **conversation\_id** | String | No | Session ID. If not provided, a new session is created. To continue an existing session, pass the `conversation_id` from the stream output. |
| **model** | String | Yes | Default model: `glm-4-assistant` |
| **stream** | Boolean | Yes | Only supports streaming SSE. Must be set to `True`. |
| **messages** | List<Object> | Yes | Session message body |
| **role** | String | Yes | User input role: `role = user` |
| **content** | List<Object> | Yes | Session message body |
| **type** | String | Yes | Currently supports `type=text` |
| **text** | String | Yes | User-input text content |
| **metadata** | Object | No | Metadata, extended fields |

### Request Example

```
from zhipuai import ZhipuAI  
api_key = "Your API key"  
url = "https://open.bigmodel.cn/api/paas/v4"  
client = ZhipuAI(api_key=api_key, base_url=url)  
generate = client.assistant.conversation(  
    assistant_id="659e54b1b8006379b4b2abd6",  
    conversation_id=None,  
    model="glm-4-assistant",  
    messages=[  
        {  
            "role": "user",  
            "content": [{  
                "type": "text",  
                "text": "Comprehensively analyze the impact of Q1 2025 Middle East geopolitical conflicts on the global energy market, incorporating data on crude oil price fluctuations, policy adjustments by major oil-producing countries, and Europe's energy substitution plans. Generate a timeline-based risk assessment report, highlighting real-time reactions in futures markets and key recommendations from institutions like the IEA."  
            }]  
        }  
    ],  
    stream=True,  
    attachments=None,  
    metadata=None  
)  

for resp in generate:  
    print(resp)
```
