Search-Std (ZhiPu Self-developed Basic Search Version) will be offered for free until May 31, 2025. Starting from 0:00 on June 1, 2025, the price will be adjusted to 0.01 yuan per search.

BigModel offers developers a comprehensive suite of AI-powered search tools, spanning **three tiers: basic retrieval (Web Search API), Q&A enhancement (Web Search in Chat), and search agents (Agent Search)**. These tools integrate proprietary engines and third-party services (Sogou/Quark) via a unified API, providing end-to-end capabilitiesâ€”from raw web data scraping and fusion of search results with LLM-generated content to multi-turn conversation context management. This empowers developers to build **trustworthy, real-time, and traceable** AI applications at **lower costs**.

### Product Overview

| Service Module | Developer Value | Technical Features |
| --- | --- | --- |
| **Web Search API** | Direct access to **structured search results** (title/content/URL, etc.) | Multi-search engine support |
| **Chat Search** | Integrates search results into LLM-generated **answers with cited sources** | Seamless real-time retrieval + LLM synthesis |
| **Agent Search** | **Query decomposition** based on intent, with dialog state management and smart routing | Intent-aware context management and engine routing |

#### Web Search API

Combining ZhiPuâ€™s self-developed models for vectors, semantic matching, timeliness, and content quality, we identify the userâ€™s intent in their questions and optimize the extraction of Query search terms. We call upon mainstream search engines (ZhiPuâ€™s self-developed, Sogou, Quark) and simultaneously use our self-developed semantic reranker to analyze the relevance of the userâ€™s question to the given search results, incorporating timeliness to provide scoring and ranking. This process delivers accurate and reliable structured webpage data to users (including titles, summaries, source links, etc.).

* **API Docs**: [Web Search API](https://bigmodel.cn/dev/api/search-tool/web-search)
* **Example**: Query recent financial news

```
from zhipuai import ZhipuAI

client = ZhipuAI(api_key="")  # Fill in your own APIKey

response = client.web_search.web_search(
   search_engine="search-pro",
   search_query="search economic events",
   count=15, # The number of results to return, ranging from 1-50, default 10
   search_domain_filter="www.sohu.com", # Only access content from specified domain names.
   search_recency_filter="noLimit", # Search for content within specified date ranges
   content_size="high" # Control the word count of webpage summaries, default medium
)
print(response)
```

#### Web Search in Chat

Enables real-time web search within the Completions API, combining retrieved results with GLMâ€™s generative capabilities to deliver up-to-date, verifiable answers.

messages = [{  
    "role": "user",  
    "content": "Important financial events in April 2025"  
}]  

# Call the API  
response = client.chat.completions.create(  
    model="glm-4-air",  # Model identifier

print(response)

#### Agent Search

Decomposes complex queries into sub-searches, aggregates results, and generates in-depth summaries with intent-aware context management.

* **API Docs**: [Agent Search](https://bigmodel.cn/dev/api/search-tool/agent-search)
* **Example**:

```
from zhipuai import ZhipuAI
client = ZhipuAI(api_key="YOUR_KEY", base_url="https://open.bigmodel.cn/api/paas/v4")
generate = client.assistant.conversation(
    assistant_id="your_assistant_id",
    model="glm-4-assistant",
    messages=[{
        "role": "user",
        "content": "Analyze Q1 2025 Middle East geopolitical impacts on global energy markets..."
    }],
    stream=True
)
for resp in generate:
    print(resp)  # Streamed response with multi-query decomposition
```
