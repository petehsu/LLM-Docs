# 智能表盘设计agent

> 一款模拟面向智能手表厂商和开发者的AI驱动表盘设计agent工具，通过对话式交互快速生成各类风格的手表表盘UI。基于MiniMax-M2大模型的Code Agent架构，让表盘设计快速高效

通过MiniMax-M2驱动的Code Agent技术，让智能手表表盘的创作像聊天一样简单，支持快速推出独特、高质量的表盘设计，最终为用户提供更丰富、更个性化的产品体验。

### **项目功能**

**1. Code Agent架构**

不是配置工具，而是真正的代码创作智能体，具备理解、生成、编辑代码的完整能力。

**2. 对话式创作流程**

将传统的"需求文档→设计稿→开发实现"流程，压缩为"对话交互→即时生成"，效率提升百倍。

**3. 零门槛专业化**

让产品经理、运营人员、设计师都能直接创作专业级表盘，打破技术壁垒。理解设计需求，从空白开始编写完整的表盘代码
