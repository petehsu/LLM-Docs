## Models in detail

## Llama-4-Maverick-17B-128E-Instruct-FP8

Model ID: `Llama-4-Maverick-17B-128E-Instruct-FP8`An industry-leading natively multimodal model that offers image and text understanding with groundbreaking intelligence and fast responses at a low cost.

| Context window | Input modalities | Output modalities |
| --- | --- | --- |
| 128k tokens | Text, image | Text |

## Llama-4-Scout-17B-16E-Instruct-FP8

Model ID: `Llama-4-Scout-17B-16E-Instruct-FP8`A class-leading natively multimodal model that offers superior text and visual intelligence.

| Context window | Input modalities | Output modalities |
| --- | --- | --- |
| 128k tokens | Text, image | Text |

## Llama-3.3-70B-Instruct

Model ID: `Llama-3.3-70B-Instruct`A text-only instruction-tuned model with enhanced performance relative to Llama 3.1 70B, and relative to Llama 3.2 90B when used for text-only applications.

| Context window | Input modalities | Output modalities |
| --- | --- | --- |
| 128k tokens | Text | Text |

## Llama-3.3-8B-Instruct

Model ID: `Llama-3.3-8B-Instruct`A lightweight and ultra-fast variant of Llama 3.3 70B, for use when quick response times are needed most.

| Context window | Input modalities | Output modalities |
| --- | --- | --- |
| 128k tokens | Text | Text |
