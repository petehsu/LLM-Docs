# Registration and Login Issues

### Does the platform support registration with overseas phone numbers?

Our platform supports registration with overseas phone numbers. During the registration process, please select the corresponding country code and verify through an SMS verification code to complete the registration.

### What are the reasons for not receiving the SMS verification code during registration/login?

To ensure the security of user accounts, we have set limits on the frequency of sending verification codes. If a verification code is sent frequently in a short period, it may trigger this limit, leading to a failed verification code delivery. Please try to receive the verification code again after 5 minutes. If the above method does not solve the problem, please contact customer service.

### How to Register as a Corporate Account?

Thank you for your use. Please complete the enterprise real-name authentication on the [Real-Name Authentication](https://bigmodel.cn/usercenter/auth)  page to become a corporate account.

### Can the same account be logged in on multiple computers simultaneously?

Thank you for your use. The Open Platform account supports simultaneous login on multiple devices, and there is currently no limit on the number of login endpoints.
