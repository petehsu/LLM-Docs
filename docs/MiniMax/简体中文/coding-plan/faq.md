# 常见问题

> 参考文档，了解关于 Coding Plan 编程套餐的订阅相关问题

### **Q: "每 5 小时重置" 是如何计算的?**

A: 这是一个动态的速率限制。系统会计算您在 5 小时内的 prompt 总用量。您在 5 小时前使用的额度会自动释放。

***

### **Q: 为什么 "一个 prompt 约等于 15 次模型调用"？**

A: 在 AI 编程工具中，您的一次操作（例如请求代码补全或解释代码）在工具后台可能会被拆分为多次与 AI 模型的连续交互（例如：获取上下文、生成建议、修正建议等）。 为了简化计费，我们将这些后台的连续调用打包为一次 "prompt" 计数。这意味着您在套餐内的一次 "prompt" 实际上包含了多次模型的复杂调用。

***

### **Q: 达到限额上限怎么办?**

A: 当您在 5 小时周期内达到 prompt 使用上限时，您可以选择以下几种方式：

* 切换到按量付费（Pay as you go） <br />
  如果您希望不受速率限制，您可以将编程工具中的 API Key 更换为您的从账户管理系统，MiniMax 开放平台普通的 API Key。这样，工具将切换到按实际 token 使用量计费的模式，消耗您的开放平台账户余额。

* 等待重置 <br />
  Code Plan 的限额是基于一个动态的 5 小时窗口。您可以暂停使用，等待窗口滚动，下一个时间的额度将会自动恢复。

***

### **Q: Coding Plan 的 API Key 和开放平台普通的 API Key 在文本模型的使用上可以混用吗？**

A: 不可以。

* Coding Plan API Key：专用于 Coding Plan 套餐，将遵循 "prompt" 数量和 5 小时限额计算。
* 其他开放平台的 API Key：用于所有文本 AI 服务（包括在编程工具中切换到按量付费），按实际 token 消耗量计费，消耗您的账户余额。

***

### **Q: 是否可以同时在多个工具中使用我的订阅套餐吗？**

A: 可以，您可以在所有支持的工具中使用同一订阅套餐，但额度是共享的，所有工具的使用会消耗同一套餐额度。

***

### **Q: 订阅套餐后可以退款吗？**

A: 订阅服务一经购买即视为确认，不支持退款。即使您未使用完套餐，费用也无法退回。我们建议您根据使用需求选择合适的订阅套餐和周期。

***

如您在订阅服务中遇到任何问题，可扫描底部官方客服二维码，获得支持。
