# Coding Plan 概要

> MiniMax 最新上线的 Coding Plan 是专为 AI 编码场景提供的订阅计划。

欢迎了解订阅 MiniMax 最新推出的 **Coding Plan**！

**MiniMax M2** 的强大效能，是我们实现 Intelligence with Everyone这一目标的关键一步。
为了向所有开发者提供更友好且易用的服务，我们提供了这项 Coding Plan订阅服务，包月制以1/10的Claude对应plan价格，获得满血用量体验。

## 核心优势

1. **支持最新 Minimax M2 模型**
   <br />所有 **Coding Plan** 套餐均搭载 **Minimax M2** 模型。这是一款专为Agent和编程任务打造的模型，其高效设计带来了更低的延迟和极具竞争力的成本效益。

2. **极具性价比的订阅模式**
   <br />**Coding Plan** 专为高频编程场景设计，提供高性价比的订阅模式。相比按 Token 计费，您可通过固定订阅费获取大量在编程工具中 prompt 用量，告别按 Token 计费的焦虑，拥抱专注编码的乐趣。
