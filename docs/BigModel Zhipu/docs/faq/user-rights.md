# 用户权益问题

> 用户权益问题解答

## 用户权益

#### 如何查看我的用户权益等级？

您可以在 [用户权益](https://bigmodel.cn/usercenter/corporateequity) 查看自己的积分与用户权益等级。

#### 用户权益逻辑生效时间是什么时候？

本逻辑从2025.4.7日晚22:00开始生效

#### 消耗资源包中的 Tokens 会增加积分么？

不会，仅通过消耗现金余额、或三方支付购买产品时，才能产生积分。

#### 使用Batch API 会增加积分么？

若Batch API调用时消耗的是现金余额（不含赠金），则会积累积分。但用户权益升级后的折扣对Batch API不生效。

#### 阶梯定价与用户权益的生效优先级是什么？

若推理模型配置了阶梯定价，则会以阶梯定价的逻辑生效。用户权益的折扣比例不对配置阶梯定价的模型实际生效。
