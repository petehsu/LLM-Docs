# Overview

The Zhipu AI Open Platform offers a variety of models, including language models, reasoning models, large-multimodal-models, hyper-humanlike models, and large vector models.

## Language Models

| Model | Description |
| --- | --- |
| GLM-4-Plus | **High-Intelligence Flagship**: Comprehensive performance enhancement with significantly improved capabilities for long texts and complex tasks |
| GLM-4-Air-250414 | **High Cost-Effectiveness**: The model that offers the best balance between reasoning capabilities and price |
| GLM-4-Long | **Ultra-Long Input**: Specifically designed for handling extremely long texts and memory-intensive tasks |
| GLM-4-AirX | **Ultra-Fast Inference**: Featuring extremely fast reasoning speed and powerful reasoning performance |
| GLM-4-FlashX-250414 | **High-Speed, Low-Cost**: Flash-enhanced version, featuring ultra-fast inference speed and higher concurrency guarantees |
| GLM-4-Flash-250414 | **Free Model**: Zero-cost access to the language model |

## Reasoning Models

| Model | Description |
| --- | --- |
| GLM-Z1-Air | **Reasoning Model**: Equipped with strong reasoning capabilities, suitable for tasks requiring deep reasoning |
| GLM-Z1-AirX | **Ultra-Fast Inference**: Featuring extremely fast inference speed (up to 200 tokens/s) |
| GLM-Z1-FlashX | **High-Speed, Low-Cost**: Flash-enhanced version with ultra-fast inference speed and even greater concurrency guarantees |
| GLM-Z1-Flash | **Free Inference**: Free inference API, enabling zero-cost access to large reasoning models |

## Multimodal Models

| Model | Description |
| --- | --- |
| GLM-4V-Plus-0111 | **Image/Video Understanding**: Equipped with outstanding multimodal understanding capabilities, capable of processing up to 5 images concurrently and supporting video content understanding of up to 2 hours in length |
| GLM-4V-Flash | **Free Model**: Supports single-image understanding, suitable for scenarios requiring basic image analysis |

| Model | Description |
| --- | --- |
| CogVideoX-3 | **Video Generation**:Significant improvements in image quality, stability, and physical realism simulation |
| CogVideoX-2 | **Video Generation**: Easily create videos using text or images as input |
| CogVideoX-Flash | **Video Generation**: A free and user-friendly video generation model with powerful video generation capabilities |
| CogView-4 | **Image Generation**: Equipped with strong capabilities in complex semantic alignment and instruction following, supports both Chinese and English input of any length, and can generate images in various resolutions. It also possesses robust text generation capabilities |
| CogView-3-Flash | **Free Model**: A free image generation model |

## Real-Time Audio and Video

| Model | Description |
| --- | --- |
| GLM-Realtime | **Real-Time Video Call**: Capable of real-time reasoning across text, audio, and video for smooth conversations |
| GLM-4-Voice | **End-to-End Speech Model**: GLM-4-Voice can directly understand and generate Chinese and English speech, enabling real-time voice conversations |

## Vector Models

| Model | Description | Maximum Input | Vector Dimension |
| --- | --- | --- | --- |
| Embedding-3 | **Latest Model**: Supports custom vector dimensions | 8K | 2048 |
| Embedding-2 | **Old Model**: Has been replaced by Embedding-3 | 8K | 1024 |

## Other Models

| Model | Description |
| --- | --- |
| CharGLM-4 | **Humanlike Model**: Suitable for emotional companionship and virtual characters |
| Emohaa | **Psychological Model:** Possesses professional counseling capabilities, helps users understand emotions and cope with emotional problems |
| CodeGeeX-4 | **Code Model**: Suitable for code auto-completion tasks |
| Rerank | **Reranking Model**: Computes score values between texts and reranks the retrieval results |

## Models Scheduled for Deprecation

We have announced the deprecation dates for the following models. After these models are deprecated, we will automatically route them to the new models. Please note that before the deprecation date, update your model encoding to the latest version to ensure a smooth transition of services.

| --- | --- | --- |
| GLM-3-Turbo | 2025-06-01 | GLM-4-Air-250414 |
| GLM-4-0520 | 2025-12-30 | - |
| GLM-4 | 2025-06-30 | - |
| CogView-3 | 2025-03-30 | - |
